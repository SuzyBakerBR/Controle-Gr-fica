# 🔑 Como Criar Token de Acesso GitHub

## Passo 1: Acesse as configurações
1. Entre no GitHub.com
2. Clique na sua foto de perfil (canto superior direito)
3. Clique em **Settings** (Configurações)

## Passo 2: Criar Token
1. No menu da esquerda, clique em **Developer settings**
2. Clique em **Personal access tokens**
3. Clique em **Tokens (classic)**
4. Clique no botão **Generate new token**

## Passo 3: Configurar o Token
1. **Note (nome):** Digite "Meu PC"
2. **Expiration:** Escolha "No expiration" (ou 90 dias)
3. **Scopes:** Marque ✅ **repo** ( primeiro item)

## Passo 4: Gerar
1. Clique em **Generate token**
2. **IMPORTANTE:** Copie o token mostrado (é uma longa sequência de letras e números)
3. **Cole o token** no terminal quando pedir a senha

---

## ⚠️ O token é muito importante:
- Parece algo como: `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- Você só vê **uma vez** após criar!
- Se perder, precisa criar outro

---

## No terminal:
Quando pedir "Password", cole o token (não vai aparecer os caracteres, é normal) e pressione Enter.
